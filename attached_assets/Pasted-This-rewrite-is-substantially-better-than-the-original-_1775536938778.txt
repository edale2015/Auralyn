This rewrite is substantially better than the original, and the five bugs it identifies are real. I would treat it as a strong midway fix, not the final vault door.

The biggest remaining issues are these:

stableStringify is still not fully deterministic
It sorts object keys, but it does not recursively canonicalize objects inside arrays. An array like [{b:1,a:2}] will still depend on insertion order because it falls back to JSON.stringify(array). It also behaves badly with undefined, because JSON.stringify(undefined) returns undefined, not a JSON string.

The mutex can poison itself after one failure
With:

chainMutex = chainMutex.then(async () => { ... });
await chainMutex;

if one write fails, chainMutex becomes rejected. After that, every future call chains onto a rejected promise and may never run the write block. One DB hiccup and the queue turns into a brick.

This is only safe in one Node process
The promise mutex protects a single process only. If you run multiple instances, workers, containers, or PM2 cluster mode, two processes can still race and fork the chain. For production, this needs a database lock inside a transaction, not just in-memory serialization.
createdAt DESC is not a safe source of truth for chain order
Two writes can share the same timestamp, or timestamps can drift. The chain should be ordered by a monotonic primary key like id BIGSERIAL, not just createdAt.
The comments say “transaction”, but there is no transaction
The code computes and inserts in sequence, but not inside an actual DB transaction. For an audit chain, words and reality should match perfectly, like scalpels and fingers.
You still need whole-chain verification
verifyChainLink() checks one link. For regulated audit posture, you also want a function that walks the entire chain and proves there are no gaps, mismatches, or forks.

Here is the version I would actually move toward.

// server/audit/hashChain.ts
import crypto from "crypto";

function canonicalize(value: unknown, inArray = false): string {
  if (value === null) return "null";

  if (value === undefined) {
    return inArray ? "null" : "";
  }

  const t = typeof value;

  if (t === "string" || t === "number" || t === "boolean") {
    return JSON.stringify(value);
  }

  if (Array.isArray(value)) {
    return "[" + value.map(v => canonicalize(v, true)).join(",") + "]";
  }

  if (t === "object") {
    const obj = value as Record<string, unknown>;
    const keys = Object.keys(obj)
      .filter(k => obj[k] !== undefined)
      .sort();

    return "{" + keys.map(k =>
      JSON.stringify(k) + ":" + canonicalize(obj[k], false)
    ).join(",") + "}";
  }

  return JSON.stringify(String(value));
}

export function stableStringify(value: unknown): string {
  return canonicalize(value, false);
}

export function computeChainHash(
  prevHash: string,
  entry: Record<string, unknown>
): string {
  const content = prevHash + stableStringify(entry);
  return crypto.createHash("sha256").update(content, "utf8").digest("hex");
}

export function verifyChainLink(
  entry: Record<string, unknown>,
  prevHash: string,
  claimedHash: string
): boolean {
  try {
    if (!/^[a-f0-9]{64}$/i.test(claimedHash)) return false;

    const expected = computeChainHash(prevHash, entry);
    const expectedBuf = Buffer.from(expected, "hex");
    const claimedBuf = Buffer.from(claimedHash, "hex");

    if (expectedBuf.length !== claimedBuf.length) return false;
    return crypto.timingSafeEqual(expectedBuf, claimedBuf);
  } catch {
    return false;
  }
}
// server/audit/auditLogger.ts
import { db } from "../db";
import { auditLogs } from "../../shared/schema";
import { eq, asc, desc, sql } from "drizzle-orm";
import { v4 as uuidv4 } from "uuid";
import { computeChainHash } from "./hashChain";

export function createTraceId(): string {
  return uuidv4();
}

// Failure-safe queue: one rejection does not permanently break the chain
let chainQueue: Promise<unknown> = Promise.resolve();

function enqueueExclusive<T>(fn: () => Promise<T>): Promise<T> {
  const run = chainQueue.then(fn, fn);
  chainQueue = run.then(() => undefined, () => undefined);
  return run;
}

export async function auditStep({
  traceId,
  step,
  input,
  output,
  metadata = {},
}: {
  traceId: string;
  step: string;
  input: unknown;
  output: unknown;
  metadata?: Record<string, unknown>;
}): Promise<void> {
  await enqueueExclusive(async () => {
    await db.transaction(async (tx) => {
      // PostgreSQL advisory lock: protects across app instances too
      await tx.execute(sql`select pg_advisory_xact_lock(91424017)`);

      const last = await tx
        .select({
          id: auditLogs.id,
          hash: auditLogs.hash,
        })
        .from(auditLogs)
        .orderBy(desc(auditLogs.id))
        .limit(1);

      const prevHash = last.length ? last[0].hash : "GENESIS";

      const entry: Record<string, unknown> = {
        traceId,
        step,
        input: input ?? null,
        output: output ?? null,
        metadata: metadata ?? {},
      };

      const hash = computeChainHash(prevHash, entry);

      await tx.insert(auditLogs).values({
        traceId,
        step,
        input: input ?? null,
        output: output ?? null,
        metadata: metadata ?? {},
        prevHash,
        hash,
      });
    });
  });
}

export async function getTraceSteps(traceId: string) {
  return db
    .select()
    .from(auditLogs)
    .where(eq(auditLogs.traceId, traceId))
    .orderBy(asc(auditLogs.id));
}

export async function verifyEntireChain(): Promise<{
  ok: boolean;
  brokenAtId?: number;
  reason?: string;
}> {
  const rows = await db
    .select()
    .from(auditLogs)
    .orderBy(asc(auditLogs.id));

  let expectedPrev = "GENESIS";

  for (const row of rows) {
    if (row.prevHash !== expectedPrev) {
      return {
        ok: false,
        brokenAtId: row.id,
        reason: "prevHash mismatch",
      };
    }

    const entry = {
      traceId: row.traceId,
      step: row.step,
      input: row.input ?? null,
      output: row.output ?? null,
      metadata: row.metadata ?? {},
    };

    const recomputed = computeChainHash(row.prevHash, entry);

    if (recomputed !== row.hash) {
      return {
        ok: false,
        brokenAtId: row.id,
        reason: "hash mismatch",
      };
    }

    expectedPrev = row.hash;
  }

  return { ok: true };
}

The most important fixes are the failure-safe queue, transaction + advisory lock, array-safe canonicalization, and ordering by id instead of createdAt.
