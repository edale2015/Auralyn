🔁 PART 1 — REAL-TIME CONTROL ENGINE

This runs continuously and makes micro-adjustments.

📁 server/control/adaptiveController.ts
type SystemState = {
  revenuePerHour: number;
  denialRate: number;
  waitTime: number;
  capacity: number;
};

type ControlAction = {
  pricingAdjustment?: number;
  routingBias?: string;
  intakeLimit?: number;
};

export class AdaptiveController {
  decide(state: SystemState): ControlAction {
    const action: ControlAction = {};

    // Revenue optimization
    if (state.revenuePerHour < 1000) {
      action.pricingAdjustment = 1.1;
    }

    // Denial control
    if (state.denialRate > 0.1) {
      action.routingBias = "low-risk-payers";
    }

    // Wait time control
    if (state.waitTime > 30) {
      action.intakeLimit = 0.8;
    }

    return action;
  }
}
🎛️ PART 2 — CONTROL POLICY LAYER

Defines what the system is allowed to change (very important in healthcare).

📁 server/control/controlPolicy.ts
export class ControlPolicy {
  enforce(action: any) {
    // Hard limits
    if (action.pricingAdjustment > 1.2) {
      action.pricingAdjustment = 1.2;
    }

    if (action.intakeLimit < 0.5) {
      action.intakeLimit = 0.5;
    }

    return action;
  }
}
🧠 PART 3 — MULTI-OBJECTIVE OPTIMIZER

Balances competing goals:

💰 Revenue

🛡️ Safety

😊 Patient experience

📁 server/control/multiObjectiveOptimizer.ts
export class MultiObjectiveOptimizer {
  score(state: any) {
    const revenueScore = state.revenuePerHour / 1000;
    const safetyScore = 1 - state.denialRate;
    const experienceScore = 1 - state.waitTime / 60;

    return (
      0.5 * revenueScore +
      0.3 * safetyScore +
      0.2 * experienceScore
    );
  }
}
⚡ PART 4 — STREAMING EVENT LOOP

This turns your system into real-time mode.

📁 server/control/controlLoop.ts
import { AdaptiveController } from "./adaptiveController";
import { ControlPolicy } from "./controlPolicy";

const controller = new AdaptiveController();
const policy = new ControlPolicy();

export async function runControlLoop(getState: any, applyAction: any) {
  setInterval(async () => {
    const state = await getState();

    let action = controller.decide(state);
    action = policy.enforce(action);

    await applyAction(action);

    console.log("Control action applied:", action);

  }, 5000); // every 5 seconds
}
🛡️ PART 5 — SAFETY ENVELOPE (CRITICAL)

Never let the system drift into unsafe territory.

📁 server/control/safetyEnvelope.ts
export class SafetyEnvelope {
  validate(state: any) {
    if (state.denialRate > 0.2) {
      return { safe: false, reason: "High denial rate" };
    }

    if (state.waitTime > 60) {
      return { safe: false, reason: "Excessive wait time" };
    }

    return { safe: true };
  }
}

export async function runFullClosedLoop(stateProvider: any, actuator: any) {
  const state = await stateProvider();

  const safety = new SafetyEnvelope().validate(state);

  if (!safety.safe) {
    console.log("System paused:", safety.reason);

  const optimizer = new MultiObjectiveOptimizer();

  await actuator(action);

    score,
    action
  };
}
