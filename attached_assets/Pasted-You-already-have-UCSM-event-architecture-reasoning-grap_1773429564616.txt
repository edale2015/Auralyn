1. Temporal Symptom Progression Engine

This engine tracks how symptoms evolve over time, which is extremely important diagnostically.

Example:

Day 1: sore throat
Day 3: fever
Day 5: shortness of breath


This progression suggests pneumonia more strongly than viral URI.

File
server/temporal/temporalSymptomEngine.ts

Code
export function analyzeTemporalProgression(events:any[]){

  const timeline:any[] = []

  for(const e of events){

    if(e.type === "PATIENT_MESSAGE"){

      timeline.push({
        timestamp:e.timestamp,
        text:e.payload.message
      })

    }

  }

  const progression:any = {}

  for(const t of timeline){

    const text = t.text.toLowerCase()

    if(text.includes("fever"))
      progression.fever = true

    if(text.includes("shortness of breath"))
      progression.sob = true

    if(text.includes("cough"))
      progression.cough = true

  }

  return progression

}

2. Patient Risk Profile Engine

Certain patient characteristics dramatically change diagnostic probability.

Example:

elderly + cough + fever → pneumonia risk
young + sore throat → viral

File
server/risk/patientRiskProfileEngine.ts

Code
export function computePatientRiskProfile(patient:any){

  const risk:any = {}

  if(!patient) return risk

  if(patient.age > 65)
    risk.ageRisk = 0.3

  if(patient.pmh?.includes("diabetes"))
    risk.metabolicRisk = 0.2

  if(patient.pmh?.includes("copd"))
    risk.respiratoryRisk = 0.4

  if(patient.pregnant)
    risk.pregnancyRisk = 0.4

  return risk

}


This risk vector feeds into differential weighting.

3. Physician Feedback Learning Engine

Doctors can correct the system:

System: viral URI
Doctor: pneumonia


The system learns from that.

File
server/learning/physicianFeedbackEngine.ts

Code
import fs from "fs/promises"
import path from "path"

const FILE = path.resolve(
  process.cwd(),
  "server/data/runtime/physician_feedback.json"
)

export async function recordPhysicianFeedback(data:any){

  let rows:any[] = []

  try{
    rows = JSON.parse(await fs.readFile(FILE,"utf8"))
  }catch{}

  rows.push({
    ...data,
    timestamp:new Date().toISOString()
  })

  await fs.writeFile(
    FILE,
    JSON.stringify(rows,null,2)
  )

}

4. Diagnostic Consensus Engine

This merges signals from multiple engines.

Signals include:

File

Code
export function computeDiagnosticConsensus(params:any){

  const votes:Record<string,number> = {}

  for(const source of params.sources){

    for(const d of source){

      votes[d.diagnosis] =
        (votes[d.diagnosis] ?? 0) + d.score

    }

  }

  return Object.entries(votes)
    .map(([diagnosis,score])=>({
      diagnosis,
      score
    }))
    .sort((a,b)=>b.score-a.score)

}

import { loadOutcomeMemory }
from "../similarity/outcomeCaseMemory"

export async function trainOutcomeModel(){

  const rows =
    await loadOutcomeMemory()

  const stats:any = {}

  for(const r of rows){

    const dx =
      r.actualDiagnosis

    stats[dx] ??= {
      count:0
    }

    stats[dx].count++

  }

  return stats

}

7. Confidence Calibration Engine

Improves diagnostic probability accuracy over time.

File
server/diagnostics/confidenceCalibrationEngine.ts

Code
export function calibrateConfidence(differential:any[]){

  const total =
    differential.reduce(
      (s,d)=>s+d.score,
      0
    )

  return differential.map(d=>({

    diagnosis:d.diagnosis,
    probability:
      d.score / total

  }))

}


This converts raw scores into probabilities.
