The target design
One generic pipeline for every complaint

Every complaint runs the same stages:

Core questions already define answers.*

ScoreDefs computes scores.* (Centor, Wells-ish, CURB-65, etc.)

Red flags compute redFlagGate (PASS/ESCALATE/ER_SEND + triggered list)

Disposition rules pick disposition_level + template_id

Cluster scoring rules pick ranked clusters (top1/top3)

Diff/Confidence (generic): uses evidence hits + contradictions + missing anchors

Render output templates

Everything above is driven by Sheets/CSVs you already have.
