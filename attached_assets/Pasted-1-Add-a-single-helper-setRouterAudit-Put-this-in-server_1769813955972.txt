1) Add a single helper: setRouterAudit(...)

Put this in server/flows/whatsappFlowRouter.ts (or wherever your menu helpers live).

export type RouterAuditInput = {
  routerReason: "menu" | "keyword" | "other_text";
  routerPickedFlowId: string;
  routerPickedSystem?: string;
  routerTextSnippet: string;
};

export function setRouterAudit(answersObj: any, audit: RouterAuditInput) {
  const a = answersObj || {};
  const ts = Date.now();

  // Keep your canonical schema
  a.__routerAudit = {
    routerReason: audit.routerReason,
    routerPickedFlowId: audit.routerPickedFlowId,
    routerPickedSystem: audit.routerPickedSystem || "",
    routerTextSnippet: audit.routerTextSnippet,
    ts,
  };

  // Add compatibility alias (my earlier schema)
  a.__router = {
    source: audit.routerReason,
    pickedFlowId: audit.routerPickedFlowId,
    pickedSystem: audit.routerPickedSystem || "",
    snippet: audit.routerTextSnippet,
    ts,
  };

  return a;
}

Why this is safe

Doesn’t rename anything

Adds alias every time you set audit

Works on old encounters (they just won’t have __router until next update)

2) Update webhook: replace manual assignment with setRouterAudit

Anywhere you currently do something like:

answersObj.__routerAudit = { ... }


Replace with:

A) Menu selection (source = menu)
const cleared = setMenuState(answersObj, { awaitingChoice: false, awaitingOtherText: false });
setRouterAudit(cleared, {
  routerReason: "menu",
  routerPickedFlowId: pick.flowId,
  routerPickedSystem: pick.system,
  routerTextSnippet: msg.slice(0, 60),
});

B) Keyword routing (source = keyword)
const updated = getAnswersObj(encounter.answers);
setRouterAudit(updated, {
  routerReason: "keyword",
  routerPickedFlowId: pick.flowId,
  routerPickedSystem: pick.system,
  routerTextSnippet: msg.slice(0, 60),
});

C) “Other text” routing (source = other_text)
const cleared = setMenuState(answersObj, { awaitingChoice: false, awaitingOtherText: false });
setRouterAudit(cleared, {
  routerReason: "other_text",
  routerPickedFlowId: pick.flowId,
  routerPickedSystem: pick.system,
  routerTextSnippet: msg.slice(0, 60),
});


Then you store answers: JSON.stringify(updatedOrCleared) as usual.
